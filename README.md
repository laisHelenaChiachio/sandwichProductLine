# sandwichProductLine

## Generator

Create Jar with

`mvn clean install`

Run with

`java -jar .\target\sandwichProductLine-0.0.1-SNAPSHOT.jar`

The generator uses `mvn` to generate the code. If the above does not generate the code you might need to supply the path to your maven installation:

`java -jar .\target\sandwichProductLine-0.0.1-SNAPSHOT.jar "\path\to\maven\mvn"`

## Final Software

Create using generator Jar

Run with
`java -jar .\out\target\sandwichSoftware-0.0.1-SNAPSHOT.jar`
